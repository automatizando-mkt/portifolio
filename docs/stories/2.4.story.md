## Story

**As a** visitante do portfólio,
**I want** uma página inicial impactante que apresente o profissional e seus trabalhos,
**so that** eu entenda rapidamente o que ele faz e queira ver mais.

## Acceptance Criteria

1. Hero section com headline, subtítulo, CTA principal e visual de destaque
2. Seção de proposta de valor (3-4 cards com benefícios/diferenciais)
3. Preview de cases em destaque (3 cards com link para página de Cases)
4. CTA secundário para página de Contato
5. Responsivo em mobile, tablet e desktop
6. Performance: LCP < 3s em conexão simulada 3G

## Dev Notes

### Canvas DS — Hero Pattern

- 12-col grid: `grid-cols-1 lg:grid-cols-12`
- Headline: `text-[7.5rem]` desktop, `text-6xl` mobile, weight 500, tracking-tighter
- Floating cards opcionais sobre grid background
- Text reveal com fadeSlideIn animation
[Source: docs/architecture/design-reference.md]

### Canvas DS — Button Pattern

```html
<a class="group relative inline-flex items-center p-4 border border-neutral-800
   hover:border-[#F97316] transition-colors duration-300">
  <div class="absolute top-0 left-0 w-2 h-2 border-t border-l border-[#F97316]
       opacity-0 group-hover:opacity-100 transition-all duration-300"></div>
  <button class="flex items-center gap-3 bg-transparent text-white font-medium
       uppercase tracking-wider text-sm">
    Label <ArrowIcon />
  </button>
</a>
```
Hover: background fills `#F97316`, text becomes black.
[Source: docs/architecture/design-reference.md]

### Canvas DS — Card Pattern

```html
<div class="border border-white/10 bg-[#050505] p-8 group
     hover:border-[#F97316]/30 transition-colors duration-300">
</div>
```
[Source: docs/architecture/design-reference.md]

### Canvas DS — MetricCard Pattern

```html
<div class="text-center">
  <div class="text-4xl font-light tracking-tighter text-white">+300%</div>
  <div class="text-xs font-semibold uppercase tracking-widest text-neutral-600 mt-1">Label</div>
</div>
```
[Source: docs/architecture/design-reference.md]

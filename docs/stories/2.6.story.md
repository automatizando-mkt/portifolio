### Dynamic Lucide Icons

Para renderizar ícones dinamicamente baseado no nome string, usar um map:

```typescript
import { Workflow, MessageSquare, Plug, BarChart3, Mail, Compass } from 'lucide-react';

const iconMap: Record<string, React.ComponentType<{ className?: string }>> = {
  Workflow, MessageSquare, Plug, BarChart3, Mail, Compass,
};
```
